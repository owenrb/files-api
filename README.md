# files-api

File Manager API

## Requirements

- Java JDK (17 recommended)
- Apache Maven

## Configuration

Open `application.properties` file for editing, and set the `files.root.folder` value accordingly.

## Execute

```shell
$ mvn clean spring-boot:run
```
